# 🚀 QuickCart

QuickCart is a modern **e-commerce web application** built with **Next.js** and **Tailwind CSS**.  
It is designed to provide a fast, scalable, and user-friendly shopping experience with a clean, component-driven architecture.

---

## ✨ Features
- ⚡ **Next.js Framework** – Optimized for performance with SSR & static generation.  
- 🎨 **Tailwind CSS** – Fully responsive and highly customizable styling.  
- 🧩 **Component-Based Architecture** – Reusable and modular UI components.  
- 📦 **Cart & Context Management** – State handling with React Context API.  
- 🔒 **Environment Configuration** – Secure `.env` support for sensitive data.  
- ✅ **Linting & Code Quality** – ESLint configuration for consistent code.  

---
